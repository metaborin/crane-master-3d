# 🕹️ クレーンマスター3D

ブラウザだけで遊べる、3D のクレーンゲーム（UFO キャッチャー）です。
[Three.js](https://threejs.org/) による描画と [cannon-es](https://pmndrs.github.io/cannon-es/) による物理シミュレーション、Web Audio API で合成した BGM / 効果音を、**ビルド不要の HTML 1 ファイル**にまとめています。

## 遊び方

景品（モンスターのぬいぐるみ）をアームでつかみ、左手前の **ゲットゾーン（穴）** に落とすとスコアが増えます 🎁

操作は 2 ステップです。

1. **① 左右を決める** … `▶右` ボタン（または `→` / `D` キー）を **押している間だけ**アームが右へ動きます。離すと位置が確定します。
2. **② 奥行きを決める** … `▲奥` ボタン（または `↑` / `W` キー）を押している間だけアームが奥へ動きます。離すと**自動でキャッチ**が始まります。

狙いが正確なほどつかめる確率と保持力が上がり、運搬中に滑り落ちにくくなります。

### そのほかの操作

| 操作 | 内容 |
| --- | --- |
| `斜め` / `正面` / `横` / `真上` | カメラの視点を切り替え |
| マウスドラッグ | カメラを自由に回転 |
| `REFILL` | 景品を補充 |
| 🔊 | サウンドの ON / OFF |

## 動かし方

特別なセットアップは不要です。モダンなブラウザ（Chrome / Edge / Firefox / Safari など）で `index.html` を開くだけで遊べます。

ローカルサーバー経由で開くとより確実です。例:

```bash
# Python が入っている場合
python -m http.server 8000
# → ブラウザで http://localhost:8000/ を開く
```

> ライブラリ（Three.js / cannon-es）は CDN から読み込むため、初回はインターネット接続が必要です。

## 技術スタック

- **Three.js 0.160.0** — 3D 描画・ライティング・影
- **cannon-es 0.20.0** — 景品やアームの物理演算
- **Web Audio API** — BGM と効果音をその場で合成（著作権フリー）
- ビルドツールなし（ES Modules + importmap）
